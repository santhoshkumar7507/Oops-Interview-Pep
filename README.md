# 🐍 Object-Oriented Programming (OOP) - Four Pillars in Python

This repository demonstrates the four fundamental pillars of Object-Oriented Programming (OOP) using Python.

## 🏛️ Four Pillars of OOP

1. **Encapsulation**
2. **Abstraction**
3. **Inheritance**
4. **Polymorphism**

### 🤔 Why OOP?

Object-Oriented Programming helps developers build software that is:
* **Reusable**
* **Maintainable**
* **Scalable**
* **Secure**
* **Easy to Understand**

## 📝 Interview Summary

### 1️⃣ Encapsulation
Protects data by restricting direct access.

### 2️⃣ Abstraction
Hides implementation details and exposes only essential features.

### 3️⃣ Inheritance
Allows one class to acquire properties and methods from another class.

### 4️⃣ Polymorphism
Allows the same method to behave differently based on the object.
